Storage Project

Introduction

Console application for managing information about products in a storage.

File input and output.

This project can read from a binary file and load all information about the products read from the file. After the products are read, they can be modified, removed or new products can be added by the user and can be saved to a binary file with the new changes.

Storage operations.

After opening the storage file and all products are loaded the user can add new products to different sections inside the storage, he can remove fixed amount of products or products which are about to expire and can require a log about all changes in the storage between two dates.
